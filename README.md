**Empowering the under-served with a modern STEM education.**

The Digital Education & Safety Foundation is a non-profit organization dedicated to providing accessible, high-quality STEM education and promoting safe online practices. Our mission is to bridge the digital divide by equipping individuals with the skills and knowledge needed to thrive in a technology-driven world.

## Our Core Services

- **Entrepreneurship**: We teach how to build products using entrepreneurship principles—from gathering requirements and development to marketing within the community.
- **Engineering**: We facilitate collaboration on hands-on projects involving robotics, microcontrollers, 3D printing, and more, guided by experienced hobbyists.
- **Digital Safety**: Leveraging over a decade of experience, we advocate for and educate the community on safe and responsible online activities.

## Marketplace & Programs

We host a variety of [Programs](https://digitaleducationsafety.org/#programs) and a [Marketplace](https://digitaleducationsafety.org/#marketplace) featuring apps and games designed to make STEM learning engaging and effective.

## Local Development

To set up and run the website locally for development and testing, follow these steps:

### Prerequisites

- Ruby and Bundler
- Python 3 (for serving the site)

### Installation & Build

1.  **Install dependencies:**
    ```bash
    bundle install
    ```

2.  **Build the site:**
    ```bash
    bundle exec jekyll build
    ```

3.  **Serve the site:**
    ```bash
    python3 -m http.server 8000 --directory _site
    ```
    The site will be available at `http://localhost:8000`.
